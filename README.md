# 🎬 React Movie Favorites App

A simple and interactive **React Movie App** that allows users to browse movies fetched from an external API and save their favorite movies for later viewing.

This project demonstrates the use of **React components, API integration, state management, and routing** to build a clean and functional web application.

---

## 🚀 Features

- Fetch movies from an external movie API
- Browse a list of popular movies
- Add movies to your **Favorites list**
- View all saved movies in the **Favorites section**
- Remove movies from favorites
- Clean and responsive UI
- Client-side routing using React Router
